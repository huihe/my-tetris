# Tetris Game

A simple Tetris game built with vanilla JavaScript, HTML, and CSS.

## Live Demo
Play the game here: [https://huihe.github.io/my-tetris/](https://huihe.github.io/my-tetris/)

## How to Run Locally
1.  Clone the repository:
    ```bash
    git clone https://github.com/huihe/my-tetris.git
    ```
2.  Open `index.html` in your browser.
